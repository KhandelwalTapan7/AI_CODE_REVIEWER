
# AI Code Reviewer

## Project Overview

AI Code Reviewer is a tool designed to review Python code for quality, security, and performance using advanced AI models.

## Features

* **Code Quality Check**: Analyze your code for best practices and potential improvements.
* **Security Review**: Identifies potential security vulnerabilities in the code.
* **Performance Optimization**: Helps optimize code for better performance.
* **AI Integration**: Powered by advanced AI models to help developers review code effectively.

## Installation

### Prerequisites

Make sure you have the following installed:

* Python 3.x
* pip (Python package installer)

### Steps to install

1. Clone the repository:

   ```
   git clone https://github.com/KhandelwalTapan7/AI_CODE_REVIEWER.git
   ```
2. Navigate to the project directory:

   ```
   cd AI_CODE_REVIEWER
   ```
3. Create and activate a virtual environment:

   ```
   python -m venv venv
   source venv/bin/activate  # For Linux/macOS
   .\venv\Scripts\activate  # For Windows
   ```
4. Install the dependencies:

   ```
   pip install -r requirements.txt
   ```

## Usage

Run the code quality review tool using the following command:

```
python app.py
```

This will start the tool and allow you to review your Python code.
